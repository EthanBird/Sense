# Sense v0.1.0-m1 发布说明

- 类型：M1 中文输入与布局预览（prerelease）
- 日期：2026-07-22
- Android：10+（minSdk 29，targetSdk 36）
- 包名：`io.github.ethanbird.senseime`

## 本次交付

- 按参考截图重构暗色/亮色单 Canvas 键盘：候选栏、功能栏、主键盘与底部系统栏；
- 左下角输入法图标打开 Android 系统输入法选择器；
- 右下角剪贴板图标打开最近文本面板，点按直接上屏；
- 中文模式默认开启，支持连续全拼、候选选择、空格首选上屏与 `中/英` 切换；
- 39,002 拼音键的本地紧凑词典，以及未收录整句的有界动态分词组合；
- 数字、符号、Emoji、编辑左移、隐藏键盘和语音入口；
- `versionCode 2`、`versionName 0.1.0-m1`；
- 不声明 `INTERNET` 权限。

## 已知限制

- 当前环境没有 Android 真机或模拟器，安装、启用、触控命中、真实 InputConnection、系统输入法选择器、剪贴板和横竖屏尚需真机验收；
- M1 是全拼闭环，不包含模糊音、简繁切换、持久化用户词和完整语言模型排序；
- 剪贴板历史只保存在 IME 进程内，最多 8 条；
- 语音入口交由系统语音识别 Activity，尚未实现结果回填；
- 发布资产使用临时 debug 证书签名，只用于工程验证。

## 下一门禁

在 Android 10+ 真机安装并启用 Sense，验证中文输入、候选上屏、左右底栏入口、剪贴板、不同 DPI 与横竖屏；记录帧时间和启动 Macrobenchmark 后进入下一阶段。
