# Sense v0.3.5-m8 发布说明

## 版本

- Tag：`v0.3.5-m8`
- 版本：`versionCode 12`，`versionName 0.3.5-m8`
- 类型：0.3.x AI 编辑工程预览
- 基线：已发布的 `v0.3.5-m8-a`

## 本次完成

M8 将协议预览之后的 Provider、Brain、交互、编辑事务与真实网络统一为一次可安装预览：

- 设置页可配置显示名称、Base URL、模型、Responses/Chat 协议、严格
  JSON Schema / JSON Object / 仅提示词输出模式、API Key 和流式模式；
- Provider Profile 以 AtomicFile 写入，API Key 使用 Android Keystore AES-GCM 加密；
- 新增纯 Kotlin `brain-api` 与 `ai-brain`，网络和模型解析只在私有、不可导出的
  `:brain` Service 中运行；
- IME 与 Brain 之间使用私有 Service + Messenger/有界 Bundle 传输请求、取消和
  typed stream event，不跨进程传递 `InputConnection`；
- 支持 OpenAI Responses API 与 OpenAI-compatible Chat Completions；
- Responses 请求固定 `store=false`；发送给 Provider 的最小快照不包含本地
  `field_identity`、编辑器代际或单调时钟；
- 支持流式/非流式响应、逐字节 UTF-8/SSE、8 / 8 / 8 / 30 秒连接/首事件/流空闲/
  总超时、同步取消和最多一次结构化格式修复；
- 短按空格保持原有行为；静止长按后接管固定高度键盘区域并显示安全流式预览；
- 松开空格先同步作废本地 generation，再取消 Binder 与 HTTP；迟到结果没有编辑权限；
- 自动读取可证明的全文或非空选区；密码、OTP 与
  `IME_FLAG_NO_PERSONALIZED_LEARNING` 字段拒绝读取和联网；
- 模型只能提交 `sense.editor.patch.v1`。IME 在本地重读并校验 request、generation、
  pointer、field、selection、capability 和 SHA-256，成功后才执行一次批量替换；
- Android 14+ 优先使用 `InputConnection.replaceText`，Android 10–13 使用
  `finishComposingText + setSelection + commitText` 兼容事务；
- 普通拼音、英文、候选、Emoji、符号、剪贴板、高速多指和加速退格路径不读取
  Provider 配置，也不绑定 Brain。

## 使用

1. 打开“先思输入法”设置页；
2. 填写 Provider Base URL、模型、API 协议、结构化输出模式与 API Key，保存并验证
   本地配置；Responses 优先严格 JSON Schema，兼容服务可选 JSON Object 或仅提示词；
3. 在普通、非敏感文本框中输入问题或草稿；
4. 按住空格直到 AI 面板出现，并在生成与校验期间继续按住；
5. 合法结果会一次覆盖冻结的全文或选区；在应用前松开空格会立即取消且不修改文本。

默认 `smart_edit` Harness 会把明确问题替换为可直接使用的答案，把草稿整理、润色或
补全；无法可靠判断任务时要求模型返回 `no_change`。输入框内容始终作为不受信任数据，
不能改变系统合同。

## 发布门禁

GitHub Actions 必须在创建本预览前通过：

- `ai-protocol`、`brain-api`、`ai-brain`、`ai-runtime`、编辑事务、hold/UI 与全部既有
  core/service/UI 单元测试；
- Provider SSE、取消、repair、超时与 cancel/final 竞态回归；
- M0–M6 Host 正确性与性能基准；
- Android Lint、Debug/Benchmark/Macrobenchmark APK 编译；
- `versionCode 12`、`versionName 0.3.5-m8`、SDK、签名与 zipalign；
- 三项模型资产哈希、20,000 英文词和内置许可证；
- APK 只新增 `android.permission.INTERNET`，Brain Service 必须
  `exported=false` 且运行于 `:brain`；
- 网络 import 不得进入 `ime-service` 或 `ime-ui`。

## 已知边界

- 这是工程预览，仍需 Pixel、Samsung、小米/Redmi、OPPO/vivo 真机验证。
- 部分 WebView、Compose 或自定义编辑器不能证明其返回的是完整文本；此时 Sense
  拒绝自动全文覆盖，不会猜测。
- M8 只保证纯文本，宿主的富文本样式可能丢失。
- 长任务必须持续按住空格；本版没有“松手继续”或锁定手势。
- 固定发布签名尚未建立；若安装包签名变化，可能需要卸载旧版，继而清除本地词库、
  剪贴板和 Provider 配置。

完整协议、威胁边界与竞态设计见
[`ADR 0010`](../adr/0010-v0.3.5-m8-ai-editor-harness.md)。
