# Sense v0.1.0-m0 发布说明

- 类型：M0 工程预览（prerelease）
- 日期：2026-07-21
- Android：10+（minSdk 29，targetSdk 36）
- 包名：`io.github.ethanbird.senseime`

## 本次交付

- 五模块 Android 工程：`app`、`ime-service`、`ime-ui`、`core-input`、`benchmark`；
- Sense / 先思输入法设置首页、启用入口和输入法切换入口；
- 独立 `:ime` 进程中的 `InputMethodService`；
- 单 Canvas QWERTY、候选栏、按下态、亮暗 Arctic Glass 拟态背景；
- composing、FakeDecoder、候选上屏、删除、空格、回车和 Shift；
- 纯 Kotlin reducer、解码协议和 8 个单元测试；
- 冷/热设置页启动 Macrobenchmark 源码；
- API 36 GitHub Actions 构建门禁和受限网络环境的离线 APK 验证脚本；
- 无 `INTERNET` 权限的可安装 debug-signed APK。

## 已知限制

- 当前环境没有 Android 真机或模拟器，因此安装、启用、真实 InputConnection 输入、横竖屏与 Macrobenchmark 尚未执行；
- FakeDecoder 只覆盖少量演示词，librime 计划在 M2 接入；
- M0 尚无方向 Skill、Brain、Provider、事件 Journal 与用户记忆；
- 发布资产使用临时 debug 证书签名，仅供工程验证，不可作为商店正式包。

## 下一门禁

在 Android 10+ 真机完成安装、启用、切换与真实输入；采集冷/热启动和帧指标。真机门禁通过后关闭 M0，再进入 M1 离线输入闭环。
