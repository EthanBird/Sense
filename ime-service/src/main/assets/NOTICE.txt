Sense includes compiled dictionary data from Rime “pinyin-simp” and
CC-CEDICT, plus a popularity-ranked English word list derived from
popular-english-words. The application code remains under the repository's
MIT License. The generated Chinese dictionary and bigram assets are separately
distributed under CC BY-SA 4.0 because they incorporate CC-CEDICT data. The
English word-list asset remains under the ISC License described below.

The files at ime-service/src/main/assets/pinyin_lexicon.bin and
ime-service/src/main/assets/pinyin_syllables.txt are generated by
tools/build_pinyin_lexicon.py. Changes to the source data are: simplified-field
selection for CC-CEDICT; lowercase and tone removal; u:/ü to v normalization;
candidate-level syllable-initial extraction; duplicate removal; low-weight
CC-CEDICT fallback merging; Rime-only statistical prefix indexing; exact
multi-syllable initials indexing; Sense custom phrase overlay; ranking; and
binary encoding. The 2026-07-23 M5 build parsed 122,811 CC-CEDICT entries,
imported 80,777 previously missing (pinyin, text) pairs and retained 80,525
after the per-code candidate cap, skipped 1,922 invalid or unsupported rows,
and produced 97,538 exact keys, 7,357 isolated prefix keys, 32,372 exact
initials keys and 189,031 private hybrid-input index keys (326,298 total keys;
14,616,041 bytes; SHA-256
eda69e1ff2a972f0a4ba30f4f2699ca744d1f8d62118d3fb696fe956f0b35ef6).
The private “}” namespace maps mixed full-pinyin-plus-initials spellings back to
their canonical pinyin and is an index only, not independent linguistic source
material.

The file at ime-service/src/main/assets/pinyin_bigrams.bin is generated by
tools/build_bigram_model.py from the version 3 pinyin lexicon. The generator
uses the strongest weight for each unique primary-source exact candidate text,
excludes statistical-prefix, hybrid-index, initials-index and CC-CEDICT
fallback-tier records,
accumulates adjacent Han character evidence from dictionary weights, normalizes
it into bounded scores,
and writes a sorted version 1 binary table. The 2026-07-23 M5 build retained all
46,655 observed Han bigrams (559,870 bytes; SHA-256
c3d806b2baeac31aaa2859607ab7c01399229332c3bf77758216ec62713e9220).
For a single conservative attribution boundary, Sense distributes this model
with the generated dictionary asset set under CC BY-SA 4.0. MDBG has not
endorsed Sense.

The file at ime-service/src/main/assets/english_lexicon.txt is generated by
tools/import_english_lexicon.mjs from popular-english-words 1.0.2. The importer
preserves source popularity order, keeps unique lowercase ASCII words of 1–32
letters, permits only “a” and “i” among one-letter tokens, removes a small
documented set of markup-like tokens, and selects the first 20,000 eligible
words. The resulting asset contains exactly 20,000 words (161,139 bytes;
SHA-256
1a182354bc9c944dc28a384c21dbb9a2338e93bd963c4ee33f40b033a8f55624).
It is distributed under the ISC License; the Chinese dictionary's CC BY-SA
attribution boundary does not change that license.
